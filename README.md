# gmp

Front end for [google-music-proxy](https://gmusicproxy.github.io/).

## Project setup
```
yarn
```

Create `.env`:

```
VUE_APP_PROXY_URL=<your proxy url>
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
